# arc42agenticwrite

Arc42 Write Agents - Agentric workflow for writing and generating arc42 architecture documentation.

## Agents Included

- **arc42-write**: Interactive documentation creation
- **arc42-write-from-code**: Generate documentation from codebase

## Required Skills

- **arc42-write-doc-layout**: Directory structure and conventions
- **arc42-write-s01-introduction** through **arc42-write-s12-glossary**: Section-specific skills (S1-S12)

Each skill provides templates, interactive questions, and best practices for its section.
