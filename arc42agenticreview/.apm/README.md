# arc42agenticreview

Arc42 Review Agents - Agentric workflow for reviewing arc42 architecture documentation.

## Agents Included

### Orchestrators
- **arc42-review**: Main review orchestrator
- **arc42-review-branch**: Review branch-specific changes
- **arc42-review-conflict**: Conflict analysis orchestrator

### Section Reviewers (S1-S12)
- arc42-review-s01-introduction through arc42-review-s12-glossary

### Conflict Analysis
- arc42-review-conflict-quality-strategy (S1 ↔ S4 ↔ S10)
- arc42-review-conflict-strategy-decisions (S4 ↔ S9)
- arc42-review-conflict-constraints-compliance (S2 ↔ S4/S8/S9)
- arc42-review-conflict-context-building-blocks (S3 ↔ S5)
- arc42-review-conflict-views-consistency (S5 ↔ S6 ↔ S7)
- arc42-review-conflict-concepts-decisions (S8 ↔ S9)
- arc42-review-conflict-risks-quality (S11 ↔ S1/S10)

## Required Skills

- **arc42-doc-layout**: Structure detection and delegation protocol
- **arc42-review-format**: Review modes and templates
- **arc42-orchestrator-format**: Output formats for orchestrators
