# arc42 Agentric Packages

APM packages for arc42 agentric workflows - Review and Write agents for architecture documentation.

## Packages

### arc42agenticreview
Agents for reviewing and validating arc42 architecture documentation.

```bash
apm install @arc42/arc42agenticreview
```

### arc42agenticwrite

Agents for writing and generating arc42 architecture documentation.
```bash
apm install @arc42/arc42agenticwrite
```

## Versioning

Each package is versioned independently using Semantic Versioning. Tags follow: {package-name}/v{version}
