# arc42 Agentic Packages

[APM](https://microsoft.github.io/apm/getting-started/quick-start/)-Pakete für agentische arc42-Workflows – Review- und Write-Agents für Architekturdokumentation.

Die Pakete in diesem Repository werden automatisch aus dem Quellrepository **[matthiasnissen/arc42agentic](https://github.com/matthiasnissen/arc42agentic)** abgeleitet und synchronisiert. Alle Details zu den angebotenen agentischen Workflows, deren Funktionsweise und Konfiguration findest du dort.

## Installation

Voraussetzung ist ein installierter [APM-Client](https://microsoft.github.io/apm/getting-started/quick-start/).

### arc42agenticreview

Agentischer Workflow zum Review und zur Validierung von arc42-Architekturdokumentation.

```bash
apm install matthiasnissen/arc42agentic-packages/arc42agenticreview
```

### arc42agenticwrite

Agentischer Workflow zum Schreiben und Generieren von arc42-Architekturdokumentation.

```bash
apm install matthiasnissen/arc42agentic-packages/arc42agenticwrite
```
